# Vue Home + Auth Flow

This repository hosts a Vue 3 single-page experience with a hero-focused home page, and dedicated login and registration views wired through client-side routing.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Available Scripts

- `npm run dev`: Runs the Vite dev server with hot module replacement.
- `npm run build`: Bundles the app for production (output in `dist/`).
- `npm run preview`: Serves the production build locally for verification.

## Notes

The navigation bar links to the home, login, and register screens, and each form includes lightweight client-side validation for required fields.
